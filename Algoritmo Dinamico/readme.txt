el archivo matrices.txt contiene (como se advierte por el nombre) las matrices que serán entrada del sistema
Se organizan como sigue:
	-Cada linea contiene una matriz
	-Las dimenciones de la matriz están separadas por un space
	-Se tiene que escribir cada matriz segun el orden deseado
Matriz_dinamica.py se ejecuta con Python 3
No requiere ningun parametro de entrada
La salida muestra:
	-Matrices de entrada
	-La Tabla M obtenida
	-La expresion resultado
